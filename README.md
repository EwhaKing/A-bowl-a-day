# A-bowl-a-day
2026 KING Craft 2팀 

프로젝트 소개: ‘오늘도 한 그릇’은 다양한 손님들의 독특한 주문을 해석해 알맞은 음식을 만들어 판매하는 캐주얼 분식집 운영 시뮬레이션 게임입니다. 플레이어는 손님의 말 속 힌트를 파악하여 재료를 선택하고 조리하며, 하루 매출을 관리하고 가게를 성장시켜 나갑니다.

엔진정보:
Unity 6000.4.1f1 사용

실행방법:
1. 게임 실행 파일(.exe)을 실행합니다.
2. 메인 화면에서 “게임 시작” 버튼을 클릭합니다.
3. 손님의 주문 내용을 확인합니다.
4. 주문 내용을 바탕으로 알맞은 재료를 선택하여 음식을 제작합니다.
5. 제작이 완료되면 손님에게 음식을 제공하고 수익을 획득합니다.
6. 하루 영업 종료 후 매출 결과를 확인하고 가게 업그레이드를 진행합니다.
7. 다음 스테이지로 넘어가 더욱 다양한 주문과 손님을 상대하게 됩니다.

조작법:
- 마우스 클릭 : 재료 선택 및 UI 조작
- 드래그 앤 드롭 : 재료 이동 및 조리 과정 수행
- 확인 버튼 클릭 : 음식 완성 및 제출
- ESC 키 : 일시정지 및 설정 메뉴 호출
- UI 버튼 클릭 : 상점 이용 및 업그레이드 진행

주요기능:
1. 손님 유형별 주문 해석 시스템
- 손님마다 서로 다른 말투와 주문 방식을 사용합니다.
- 플레이어는 주문 속 힌트를 분석하여 원하는 음식을 추리해야 합니다.
- 일반 손님 외에도 독특한 주문을 하는 특수 손님이 등장합니다.
2. 재료 선택 및 조리 시스템
- 다양한 재료를 조합하여 음식을 제작할 수 있습니다.
- 올바른 재료를 선택해야 주문이 성공 처리됩니다.
- 잘못된 조합을 선택하면 손님의 만족도가 감소합니다.
3. 하루 매출 정산 시스템
- 하루 영업 종료 후 판매 결과와 총 수익을 확인할 수 있습니다.
- 플레이 결과에 따라 보상과 평가가 달라집니다.
4. 업그레이드 시스템
- 획득한 수익으로 새로운 재료와 기능을 해금할 수 있습니다.
- 조리 속도 향상, 재료 추가 등 다양한 성장 요소가 존재합니다.
5. 캐주얼 UI 및 아트 스타일
- 직관적인 인터페이스를 통해 누구나 쉽게 플레이할 수 있도록 구성하였습니다.
- 아기자기한 2D 그래픽 스타일을 적용하여 편안한 분위기를 연출하였습니다.
6. 사운드 및 연출 요소
- 조리 효과음과 손님 반응 사운드를 통해 몰입감을 높였습니다.
- 상황에 따라 다양한 연출과 이벤트가 등장하도록 구성하였습니다.
